# Verified camps and sessions are aggregates of verified claims

CampFit treats verification as current reliability over required claims, not as a single manual flag on a record. A Verified Camp requires verified identity, location, description, classification, and contact or registration-path claims; a Verified Session requires verified session dates, session time or a clear reason time does not apply, eligibility, registration status, price options, and registration path, with explicit gaps when required values are unknown.
